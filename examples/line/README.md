# JasperReports line

The `line` tag creates a simple horizontal or vertical line.  


Horizontal line:  

```xml
<band height="20">
    <line>
        <reportElement x="0" y="2" width="250" height="1"/>
    </line>
</band>
```

Vertical line:  

```xml
<line>
    <reportElement x="0" y="5" width="1" height="150"/>
</line>
```
